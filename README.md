# Easy Tape E-commerce Website

This is an e-commerce website for "Easy Tape," a premium soft tape measure product line. The website is built using HTML, CSS, and Bootstrap, with minimal JavaScript for enhanced user experience.

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Modern and unique UI with a custom color scheme
- User-friendly navigation with a unique navbar design
- Product pages with detailed information, specifications, and reviews
- Legal pages including Terms & Conditions, Privacy Policy, and Cookies Policy
- SEO-optimized with proper meta tags, sitemap, and robots.txt
- GDPR-compliant cookie consent banner
- Newsletter subscription form

2. Open any HTML file in a modern web browser
3. No server setup or database is required as this is a static website
